# Static http/https server

Simple gRPC example

## Install

    $ go get github.com/sikang99/grpc-example

## Usage

refer Makefile to know how to use
	
	$ make 

compile IDL of gRPC
	
	$ make proto

build client and server programs

	$ make build

run and test
	
	$ make run
